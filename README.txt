This game has the following functionality:
    -It will get random words from either a text file or python list
    -It will add words to the python list
    -It will take user guesses as letter or words
    -It will validate the guesses against the randomly retrieved words
    -It will print varying stages of a hangman diagram
    -It will store your guesses
    -It will allow you to view the rules of the game
